# Entrega de Tareas JavaScript

Este repositorio contiene una serie de **pequeños ejercicios en JavaScript**, desarrollados como parte de las entregas de tareas del máster en programación Full Stack.  

El objetivo es practicar conceptos fundamentales de **JavaScript**, aplicando lo aprendido en clase mediante ejercicios simples pero prácticos.
